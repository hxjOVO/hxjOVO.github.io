# Xiaojie Han Personal Homepage

This is a static personal academic homepage for Xiaojie Han, Ph.D.

## Local Preview

Open `index.html` directly in a browser.

## Structure

- `index.html`: page content
- `styles.css`: visual design and responsive layout
- `script.js`: current year and dark mode toggle

## Content Included

- Current role at Columbia University Irving Medical Center
- Research profile
- Selected publications
- Training and education
- Award and presentation highlights
- Academic contact email

Private CV details such as home address, phone number, birthplace, and citizenship are intentionally not included on the public page.

## Publishing Recommendation

For this site, the simplest route is GitHub Pages:

1. Create a repository named `your-github-username.github.io`.
2. Upload `index.html`, `styles.css`, and `script.js`.
3. Open the repository Settings.
4. Go to Pages and publish from the `main` branch.

Optional next step: use NotionNext as a separate blog or notes site, linked from this homepage.
